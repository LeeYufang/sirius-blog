h1. Keyboards shortcuts

Usually when you are using a graphical modeler, you do most of the actions with the mouse. "Sirius":http://www.eclipse.org/sirius based modelers have several built-in shortcut keys that you can use to save time in your day to day workflow. Shortcut keys are commonly accessed by using the !./images/computer_key_Alt_T.png! key, !./images/computer_key_Ctrl_T.png! key, or !./images/computer_key_Shift_T.png! key in conjunction with other keys.
